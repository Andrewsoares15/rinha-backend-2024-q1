# rinha-backend-2024-q1
### Technologies
  - Kotlin
  - GraalVM
  - Spring Boot
  - Docker / Docker compose (container orchestrator)
  - Nginx (Load balancer)
  - Postgres (Database)

### Requirements
  - [Docker](https://docs.docker.com/engine/install/)

  - ### How to run
  - execute `docker-compose up`
  - **API** will be available in the URL `http://localhost:9999`
